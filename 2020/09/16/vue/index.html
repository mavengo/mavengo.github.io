Vue入门到精通

1、Vue的基本认识

英文官网
中文官网

1、Vue.js是一套构建用户界面的渐进式JavaScript框架。

2、Vue 只关注视图层， 采用自底向上增量开发的设计。

3、Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。
4、与其他前端JS框架的关联。
（1）借鉴”angular”的模板和数据绑定技术。
（2）借鉴”react”的组件化和虚拟DOM技术。
5、Vue的特点。
（1）遵循MVVM模式。
（2）编码简洁，体积小，运行效率高，适合移动/PC端开发。
（3）它本身只关注UI，可以轻松引入vue插件或其他第三方库开发项目。
6、Vue扩展插件。
（1）vue-cli:vue脚手架，下载基于vue的项目。
（2）vue-resource(axios):ajax请求。
（3）vue-router:路由。
（4）vuex:状态管理。
（5）vue-lazyload:图片依赖加载。
（6）vue-scroller:页面滑动相关。
（7）mint-ui:基于vue的UI组件库（移动端）。
（8）element-ui：基于vue的UI组件库（PC端）。

作者：尤雨溪

（4）缓存

当调用相同的方法执行多次时，后台只运行了一次，结果当然会出现多次，这说明后台有缓存，在涉及运行效率时可以用到此知识。

7、条件渲染

（1）条件渲染指令

v-if
v-else
v-show
这些指令实现标签的显示与隐藏。

(2)比较v-if与v-show

两者的区别就是v-if是通过删除标签来隐藏标签，而v-show是通过 style=”display: none;” 来隐藏标签。
如果需要频繁切换 v-show比较好。
